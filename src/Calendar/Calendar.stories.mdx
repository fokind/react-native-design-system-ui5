# Calendar

```js
import { Calendar } from 'react-native-fundmental';
```

A **Calendar** is commonly used as the contents of a **Popover** when composing a **DatePicker**. It is rarely used on its own as a standalone component. Internally it uses and accepts [Moment.js](https://momentjs.com/) date objects.
