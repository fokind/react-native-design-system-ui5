# ListItem
A ListItem is a wrapper around `TextInput` with commonly used styles and easy to make customization.

# Usage
```js
import { ListItem } from 'react-native-fundmental';

<ListItem>
  This is a ListItem
</ListItem>
```

# Props
<Props of={ListItem} />

# Type
There are three types of button mainly: `default`, `round`, and `outline`. You can pass boolean value `round={true}` for round button, `outline={true}` for outline button and nothing for default solid button.
