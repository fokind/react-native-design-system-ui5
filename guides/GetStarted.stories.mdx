# Get Started
`Fundamental React Native` Design System is a set of React Native components like Fundamental React.

# Install
Simply go to the command line and run this command.
```sh
yarn add react-native-design-system-ui5
```
You can use `yarn` or `npm` as per your choice.
```sh
npm install react-native-design-system-ui5
```
This library needs `react-native-vector-icons` so go on and install that too to get all the cool icons. Check out [Install guide](https://github.com/oblador/react-native-vector-icons#installation).

# Usage

### Step 1. Use root component.

```jsx
function App() {
  return (
    <Root />
  );
}
```

### Step 2. Use any component.

```js
// inside any file
import { Button } from 'react-native-design-system-ui5';

function HomeScreen() {
  return (
    <Button>
      Button
    </Button>
  );
}
```

And we are done!
